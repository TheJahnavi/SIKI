SIKI - Scan It Know It Documentation

Project Overview

SIKI is a mobile-first web application for scanning and analyzing product images to provide health assessments.

Table of Contents

  • Project Structure
  • Main README

Getting Started

To run the application locally:

python -m http.server 8080

Then open http://localhost:8080 in your browser.

Development

The project uses:

  • HTML5 for structure
  • CSS3 with custom properties for styling
  • Vanilla JavaScript for functionality
  • Material Design 3 principles

Testing

Run tests with:

npm test

Building

No build step is required. The application runs directly from the source files.
